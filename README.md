## 功能介绍（完整见运行截图）
管理员： 登录和注册功能，管理学生和教师信息，课程信息管理，学生课程管理，签到管理和课堂表现管理，布置作业管理，提供提醒和警告功能，方便教学管理。

学生： 注册账户，查看个人信息，选择课程，参与签到，查看课堂表现，接收作业通知，修改个人资料，方便学习和管理个人学习进度。

教师： 注册账户，管理课程信息，查看和管理学生信息，布置作业，查看学生签到情况，记录课堂表现，提供教学反馈，便于高效授课和学生管理。
